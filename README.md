## Informações da aplicação
Aplicação Web desenvolvida em Java, componentes provisionados para o ambiente da aplicação:
<ul> 
    <li>S3 bucket</li>
    <li>VPC</li>
    <li>RDS - MySQL database</li>
    <li>ElastiCache</li>
    <li>Amazon MQ</li>
    <li>BeansTalk</li>
    <li>EC2 Ubuntu instance</li>
</ul>
